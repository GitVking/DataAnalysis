# Abstract

本次作业旨在简单介绍机器学习经典算法如决策树DecisionTree，贝叶斯原理Bayes，支持向量机SVM等算法在数据分析中的应用。主要分为以下几个模块：

1. 机器学习算法原理及其python实现

3. 前端展示
